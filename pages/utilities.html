Border Utility Classes

Utility classes for adding borders to any element. Similar to Bootstrap.

.border
This div has a full border
.border-top & .border-bottom
This div has top and bottom borders
.border-0
This div has no border (border-0 overrides border)
.border-left & .border-right
This div has left and right borders
.border-start & .border-end
This div has start and end borders (RTL-aware)
Combined Borders
Combined utility classes for custom borders

Text Color Utility Classes

Utility classes for text color. Designed for optimal readability in both light and dark modes.

.text-primary

Primary color text for emphasis

<p class="text-primary">...</p>
.text-secondary

Secondary text for less emphasis

<p class="text-secondary">...</p>
.text-success

Success message or status

<p class="text-success">...</p>
.text-warning

Warning message or alert

<p class="text-warning">...</p>
.text-danger

Error message or destructive action

<p class="text-danger">...</p>
.text-info

Information or help text

<p class="text-info">...</p>
