Utility Classes

Utility classes for borders, text colors, text sizes, layout, spacing, typography, grid, and flexbox. Similar to Bootstrap's approach to utility-first CSS.

Component Utilities

Border Utilities

Utility classes for adding borders to any element. Full border, directional borders, and combined patterns.

Text Color Utilities

Utility classes for text color. Primary, secondary, success, warning, danger, and info colors with optimal readability.

Text Size Utilities

Utility classes for text size. Matches to same sizing convention as buttons (xs, sm, md, lg, xl).

Layout Utilities

Layout Utilities

Container, flexbox, and stack utilities for layout control. Includes .container, .stack, .stack-sm, and .indent-sm.

Spacing Utilities

Margin and padding utilities for spacing elements. Available from mb-0 to mb-8, mt-1 to mt-8, and pt-1 to pt-8.

Typography Utilities

Heading and text utilities for consistent typography. Includes .h1, .h2, .text-description, .text-description-sm, and .link-primary.

Display Utilities

Grid Utilities

Grid utilities for responsive layouts. Includes .grid with gap and .grid-cards for card layouts with 280px minimum width.

Flexbox Utilities

Flexbox utilities for alignment and layout control. Includes .flex, .flex-column, .flex-wrap, .stack, .stack-sm, and gap utilities.
