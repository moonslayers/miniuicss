MiniUI CSS

Strategic Minimalist UI - Buttons, Cards, Modals & Utility Classes

Buttons

Button components with variants, sizes, and states. Primary, secondary, success, warning, danger, and info colors.

Cards

Card components with header, body, and footer sections. Clean Enterprise aesthetic with generous spacing and elegant shadows.

Modals

Modal components with overlay for visual isolation. Perfect for confirmations, alerts, and information dialogs.

Utilities

Utility classes for borders and common patterns. Similar to Bootstrap's approach to utility-first CSS.
